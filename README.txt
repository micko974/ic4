**Pour lancer:
	*PHP/MYSQL:
		- stocker le fichier .Sql (ic4.sql) dans votre répertoire qui gère les BD ( par défaut: /var/lib/mysql/ )
		- mettre le dossier IC4 dans le dossier www du serveur Apache
	*Java:
		- variable url :mettre l'adresse du serveur à l'emplacement du fichier hello.php ( par défaut : url est  mis pour 127.0.0.1/IC4/hello.php )
